RESTful Routes

A table of RESTful routes

Name Path Http verb Purpose
Index /items GET List all items
New /items/new GET Show new item form
Create /items POST Create a new item, then redircet
Show /items/:id GET Show info about specific item
Edit /items/:id PUT Show edit form for one item
Withdraw /items/:id/withdraw PUT Withdraw
Deposit /items/:id/deposit PUT Deposit
Destroy /items/:id DELETE Delete a particular item, then redirect
